# HelpLembra - Sistema de Controle de Medicamentos
## Descrição
O MedLembra é um sistema desenvolvido em Java para ajudar idosos a lembrarem de
tomar seus medicamentos nos horários corretos.
## Funcionalidades
- Cadastro de pessoas (pacientes e cuidadores)
- Cadastro de medicamentos
- Registro de horários
- Relatório de adesão
## Tecnologias utilizadas
- Java 17
- ArrayList para armazenamento em memória
## Como executar
1. Clone o repositório: `git clone https://github.com/usuario/helplembra.git`
2. Compile: `javac src/*.java`
3. Execute: `java src/Main`
